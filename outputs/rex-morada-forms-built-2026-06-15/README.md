(`.dtf*` classes) + JS in a single block. To view: open the .html in any browser.
To publish: paste the whole file into a Squarespace Code Block on the page slug.

(`font-family: inherit`); DT brand colours applied (brand `#305DAB`, ink

outlines, inline error messaging. Em-dash zero.
